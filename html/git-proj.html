This is My Git Page

Working Directory -> Staging Area -> Local Repo -> Remote Repo

Starting a Project

  1. Fresh (no source yet) - git init [project name]
  2. Existing source locally - git init (while in project folder)
  3. GitHub project (Fork and clone) - git clone [GitHub clone URL]

Commit Stortcuts

  • -a: Add to Staging Area
  • -m: Add Inline Commit Message

Tracked Files

Any File That Has:

  • Been Committed
  • Been Added to the Git Index
  • Been Added to the Staging Area

To See Tracked Files - git ls-files

Recursive Add

git add .

Back Out Changes

  • To Remove from Staging - git reset HEAD [file]
  • To Remove Completely - git checkout -- [file]

If you rename a file using the bash command "mv", git will think a file has been deleted and a new one added

Use git mv to rename files. It will automatically add it to staging

Git has to be tracking a file for it to be deleted using git rm

Alias

  • To Add a new Alias - git config --global alias.[name] "[command]"
  • To Modify an Alias - [editor] ~/.gitconfig

Ignore

Create an ignore file - [editor] .gitignore

Comparisons

  • git diff
  • git difftool
  • git difftool HEAD compares the current working directory with the last commit
  • git difftool --staged HEAD compares the staging area to the last commit
  • git difftool -- [filepath] compares changes for just one file
  • git difftool [commit 1] [commit 2] compares differences betwen commits
  • git difftool main origin/main compares local to remote repos
  • git difftool [branch 1] [branch 2] compares differences between two branches

Branches

  • To List All Branches - git branch -a
  • To Create A New Branch - git branch [branch]
  • To Switch Branches -
  • To Rename A Branch - git branch -m [branch] [new branch]
  • To Delete A Branch (Must Be In A Different Branch) - git branch -d [branch]
  • To Create and Switch To Branch - git checkout -b [branch]

Merging

  • Fast Forward Merge - (Switch To Destination Branch) git merge [source]
  • Disable Fast Forward - (Switch To Destination Branch) git merge [source] --no-ff
  • Automatic Merges - Happen when changes are made to master AND new branch
  • Dealing with Merge Conflicts - git mergetool (then do a normal commit)
